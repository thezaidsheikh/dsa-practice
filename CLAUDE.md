## Key Patterns and Approaches

### Binary Search in Rotated Arrays
- Used when a sorted array is rotated at some pivot point
- Requires finding the pivot first, then applying standard binary search
- Common problems: Find Minimum, Search in Rotated Array

### Tree Traversals
- **DFS**: Inorder (left-root-right), Preorder (root-left-right), Postorder (left-right-root)
- **BFS**: Level Order traversal with queue implementation
- **BST Operations**: Search, Insert, Delete, Lowest Common Ancestor

### Two Pointers Technique
- Used for array problems where two pointers can traverse from different ends
- Common in container with most water, trapping rain water problems

### Heap Patterns
- Max-Heap and Min-Heap implementations
- Top K problems using heap data structure
- Priority scheduling problems
